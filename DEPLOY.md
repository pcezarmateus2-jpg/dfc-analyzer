# 🚀 Deploy no Vercel - Super Simples

## Passo 1: GitHub

```bash
cd dfc-analyzer
git init
git add .
git commit -m "DFC Analyzer"
git remote add origin https://github.com/SEU-USUARIO/dfc-analyzer.git
git push -u origin main
```

## Passo 2: Vercel

1. Acesse: https://vercel.com
2. Login com GitHub
3. New Project
4. Import `dfc-analyzer`
5. Deploy!

**Pronto! Sistema online em 2 minutos!**

## 🌐 Acessar

Sua URL será algo como:
```
https://dfc-analyzer.vercel.app
```
