# 🧪 Como Testar o DFC Analyzer

## Opção 1: Teste Rápido Local (SEM Node.js)

1. Abra o arquivo `index.html` diretamente no seu navegador
2. Você verá uma interface de exemplo com dados simulados
3. **Limitação**: Não processa PDFs reais, apenas mostra exemplo

## Opção 2: Sistema Completo Online (RECOMENDADO)

Acesse: **https://dfc-analyzer.vercel.app**

O sistema está sendo implantado automaticamente no Vercel. Aguarde alguns minutos após o push.

## Opção 3: Rodar Localmente com Node.js

Se você tem Node.js instalado:

```bash
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev

# Acessar em: http://localhost:3060
```
