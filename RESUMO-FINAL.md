## 🎯 Como Usar Cada Forma

### Excel/PDF
1. Clique na aba "Excel/PDF"
2. Selecione 2 arquivos:
   - Contas a Pagar (despesas)
   - Contas a Receber (receitas)
3. Clique em "Analisar e Gerar DFC"

### Google Sheets
1. Clique na aba "Google Sheets"
2. Cole o link da planilha de Contas a Pagar
3. Cole o link da planilha de Contas a Receber
4. Clique em "Importar e Analisar"

**IMPORTANTE:** As planilhas devem estar públicas!
- Compartilhar → "Qualquer pessoa com o link"

### Entrada Manual
1. Clique na aba "Manual"
2. Cole os dados no formato:
   ```
   Descrição R$ 1.234,56
   ```
3. Clique em "Analisar Dados"

---

## 📋 Formato dos Dados

Todas as formas usam o mesmo formato:

| Loja | Conta | Valor |
|------|-------|-------|
| Matriz | Salários | 10.000,00 |
| Filial 1 | Aluguel | 2.500,00 |

## 💡 Dicas

### Para Google Sheets:
- Planilha deve estar pública
- Primeira linha = cabeçalho
- 3 colunas: Loja, Conta, Valor

### Para Excel:
- Use formato .xlsx (recomendado)
- Mesma estrutura: Loja, Conta, Valor

### Para Entrada Manual:
- Uma linha por item
- Formato: Descrição Valor

---

## ❓ Problemas?

### Google Sheets não importa
- Verifique se está público
- Use link completo (não encurtado)

### Excel não lê
- Verifique formato das colunas
- Certifique-se que valores são números

### PDF não funciona
- Use Entrada Manual como alternativa
- Copie e cole os dados
